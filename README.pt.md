## Sobre o projeto

https://rosesandmore.netlify.app/

Roses & More é uma landing page criada com o objetivo de praticar estilização interativa e layout responsivo.

A landing page consiste na apresentação breve de uma floricultura fictícia, com a interação de alterar o layout conforme a flor em destaque.
